## Unstable

[Official Docs](https://biteship.com/en/docs/intro)

### Installation
```
go get -u github.com/toel-app/biteship-go
```

### Usage
```go
package main

import (
	"fmt"
	"github.com/toel-app/biteship"
	"log"
)

func main() {
	secretKey := "abcdefg"
	biteshipApp := biteship.New(secretKey)

	resp, err := biteshipApp.GetCourier()
	if err != nil {
		panic(err)
	}
	fmt.Println(resp)
}
```
